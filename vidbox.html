VidBox

Split, merge videos and so on. Written in Vala/GTK, for Linux Desktop.

Very easy to use

Cut videos without pain

Dark mode UI

Merge any videos with one click

Feature:

  • Very easy to use
  • Lossless cutting of any video formats
  • Merge multiple videos with one click

Installation

Linux

Download AppImage from the release page.

Debian

Add DebianOpt repository, then install with sudo apt install vidbox.

Development

The source is managed on GitHub. If you encounter any bugs or want to suggest improvements, please use the issue tracker there.
