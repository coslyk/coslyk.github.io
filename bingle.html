Bingle

A simple wallpaper downloader that can download images from Bing etc.

With Bingle, you can:

  • View and download wallpapers from internet
  • Manage your local wallpapers

Bingle supports multiple websites, currently:

  • Bing

Installation

Linux

Download AppImage from the release page.

Debian

Add DebianOpt repository, then install with sudo apt install bingle.

Development

The source is managed on GitHub. If you encounter any bugs or want to suggest improvements, please use the issue tracker there.
