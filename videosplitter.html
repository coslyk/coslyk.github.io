Video Splitter

Split your videos losslessly and conveniently. Written in Vala/GTK, for Linux Desktop.

Light mode

Dark mode

Feature:

  • Very easy to use
  • Lossless cutting of most video formats
  • Cut video into multiple segments

Installation

Linux

Download AppImage from the release page.

Debian

Add DebianOpt repository, then install with sudo apt install video-splitter.

Development

The source is managed on GitHub. If you encounter any bugs or want to suggest improvements, please use the issue tracker there.
